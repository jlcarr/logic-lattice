Gate List

  • 0: Always returns 0

  • &: The AND gate
  • |: The OR gate
  • ^: The XOR gate

Instructions

Enter a pair of gates from the Gate List into the "New Gate" box.
Click a gate on the canvas to replace it with a gate of type "New Gate".
You can also click the buttons under the Sample Circuits to have a pre-build circuit loaded.
Toggling the "Animate" button changes from the instantaneous simulation to the 'falling marble' simulation. See Description below.
Toggling the "Show gate numbers" button changes from the gates on the canvas showing their operational logic symbol to their index in the array.

Description

This project takes the form of a simple webpage using HTML5 canvas and JavaScript to implement a lattice of logic gates.
Logic flow from the top to the bottom, as if the energy source for the implementation were gravity.
The 2-input 2-output logic gates are labeled by ordered pairs of 2-input 1-output logic gates we're familiar with. The left gate of the pair represent the value given to the left-output of the gate, and the right-output follows suit.

The user also has the option to toggle between the animated 'falling marble' simulation, and the more instantaneous 'electric circuit' simulation.
In the animated simulation the presence of a falling ball or lack thereof represent HIGH and LOW logic levels accordingly.
In the instantaneous simulation the red wires and blue wires represent HIGH and LOW logic accordingly.
